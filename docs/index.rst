Options Flow Analyzer Documentation
===================================

A Python-based tool for analyzing options flow and market data.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   installation

Features
--------

* Real-time options flow analysis
* Market data processing
* Configurable alerts and notifications
* Command-line interface for easy automation

Quick Start
-----------

Install the package:

.. code-block:: bash

   pip install -r requirements.txt

Run the analyzer:

.. code-block:: bash

   python -m options_flow_analyzer --help
